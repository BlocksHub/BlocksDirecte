# 📦 Installation

This package provides a Typescript API wrapper for ÉcoleDirecte.

## Requirements

- Node.js 18+ (recommended)
- Bun (recommended) or npm / yarn / pnpm

## Installation

### Using npm
```bash
npm install @blockshub/blocksdirecte
```

### Using Bun (recommended)
```bash
bun add @blockshub/blocksdirecte
```

### Using yarn
```bash
yarn add @blockshub/blocksdirecte
```

### Using pnpm
```bash
pnpm install @blockshub/blocksdirecte
```
